# Group Policy Client-Side Extension GUID List

This document lists GUIDs used by Microsoft Group Policy Client-Side Extensions (CSEs). These GUIDs often appear in Group Policy processing messages, event logs, diagnostics, and error analysis.

A **GUID** is a *Globally Unique Identifier*. A **GPO** is a *Group Policy Object*. A **CSE** is a *Client-Side Extension*, the component on a Windows client or server that processes a specific part of a Group Policy configuration.

Original German article: [GUID Liste der Gruppenrichtlinien Client Erweiterungen](https://www.infrastrukturhelden.de/microsoft-infrastruktur/active-directory/guid-liste-der-gruppenrichtlinien-client-erweiterungen/)

## Notes for troubleshooting and changes

Before changing Group Policy Objects, create a backup. This is especially important when troubleshooting policy processing issues, testing policy changes, or working in environments with several administrators.

You do not always need to back up the complete domain controller. A dedicated GPO backup and documentation process is often sufficient. Also include the **GPO Central Store**, because it contains the central ADMX and ADML policy template files used by Group Policy Management.

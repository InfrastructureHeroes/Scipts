## Beschreibung (Deutsch – maßgeblich)

In dieser Kategorie werden **alle Group Policy Object (GPO) Templates** veröffentlicht, die im Rahmen der Artikel und Projekte auf  
**InfrastructureHeroes.org** und **InfrastrukturHelden.de** entstehen.

Der inhaltliche Fokus liegt auf:

- Windows-Client- und Server-Betriebssystemen
- IT-Sicherheit, Datenschutz und Nachvollziehbarkeit
- Umsetzung von Anforderungen aus dem **BSI IT-Grundschutz**
- Kontrollierter und dokumentierter Kommunikation mit Microsoft-Cloud-Diensten
- Praxisnahen, nachvollziehbaren und prüfbaren Konfigurationen

Die bereitgestellten GPOs dienen als **Referenz- und Beispielkonfigurationen**.  
Sie stellen **keine 1:1-Vorlagen** dar und müssen vor einem produktiven Einsatz stets bewertet, getestet und an die jeweilige Zielumgebung angepasst werden.

## Description (English)

This category publishes **all Group Policy Object (GPO) templates** created as part of articles and projects on  
**InfrastructureHeroes.org**.

The focus is on:

- Windows client and server operating systems
- IT security, data protection, and traceability
- Implementation of requirements based on **German BSI IT Basic Protection**
